# 🧩 PyDex

Pokédex desktop moderna desenvolvida em Python utilizando CustomTkinter e a PokéAPI para exibir informações de Pokémon através de uma interface limpa, bonita e interativa.

---

## 📸 Preview

Uma aplicação desktop inspirada em uma Pokédex moderna com:

* Busca de Pokémon por nome ou número
* Exibição da arte oficial do Pokémon
* Navegação entre Pokémon
* Interface moderna no estilo Pokédex
* Design limpo e responsivo

---

## ✨ Funcionalidades

* 🔍 Buscar Pokémon por nome ou número
* 🖼️ Exibir imagem oficial do Pokémon
* 📖 Mostrar informações como:

  * Nome
  * Número da Pokédex
  * Tipos
  * Altura
  * Peso
  * Habilidades
* ⬅️➡️ Navegação entre Pokémon
* ⌨️ Busca utilizando a tecla Enter
* ⚠️ Tratamento de erros para Pokémon inválidos
* 🎨 Interface moderna inspirada em Pokédex

---

## 🛠️ Tecnologias Utilizadas

* Python
* CustomTkinter
* Requests
* Pillow
* PokéAPI

## 📦 Instalação

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/PyDex.git
```

Acesse a pasta do projeto:

```bash
cd PyDex
```

Instale as dependências:

```bash
pip install -r requirements.txt
```

---

## ▶️ Executando o Projeto

```bash
python main.py
```
